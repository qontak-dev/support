**Inbox - Mengirimkan Media - Web**

1. Buka [http://chat.qontak.com/](http://chat.qontak.com/ "http://chat.qontak.com/") pada browser anda (Safari, Chrome, atau Firefox) di komputer anda;

   ![](/uploads/screenshot-27.png)
